# 墨环落印起卦动效设计

**日期：** 2026-07-16  
**状态：** 用户已确认视觉方向，待实施计划审阅

## 目标

将当前起卦动画升级为层次丰富的“墨环落印”仪式：有现代中式器物感与江湖神秘感，但不使用旋转罗盘、铜钱乱飞、火焰、闪电或 3D 翻牌等廉价占卜视觉。

## 视觉构成

动效使用深墨背景、暖金密文、米白爻线和少量朱砂。朱砂只用于中心食印与动爻，不能成为大面积高饱和装饰。

1. **双层仪式边框**：结果卡内部出现深浅两层矩形边框，外层更淡；起卦完成后保留为结果卡底纹。
2. **八卦密文环**：细淡的八卦名环绕中心，不做高速旋转；只在起卦时极缓收束或淡入。
3. **时辰刻印**：报数、农历日与时辰在印环上方以小字呈现，作为“数时合参”的可见凭据。
4. **墨晕收束**：两至三层不规则/半透明墨环向中心轻收，不允许出现加载旋钮的等速循环感。
5. **朱砂食印**：中心“食”字带印谱边缘、内圈与轻微压印扩散，不做弹跳。
6. **六爻成象**：六爻严格自下向上显现，最后一爻的动爻出现朱砂细线/朱砂点。

## 时间轴

总时长 1.35 至 1.55 秒；减少动态效果时跳过所有过渡、直接呈现最终卦象和结果。

- 0–180ms：现有输入区降低对比，双边框与时辰刻印淡入。
- 180–560ms：墨环与八卦密文缓慢收束；前三爻按下至上显现。
- 560–800ms：朱砂食印短促压入，伴随小幅墨晕扩散后收止。
- 800–1120ms：余下三爻继续自下向上显现；动爻最后显朱砂。
- 1120–1500ms：印环退为结果卡底层，结果文字、餐型、具体菜建议和解释依序淡入。

## 交互与可访问性

- 起卦期间禁用起卦、重试、确认按钮，结束后恢复；
- 不依赖声音；
- `prefers-reduced-motion: reduce` 时不播放收束/压印/逐爻动画；
- 视觉装饰全部 `aria-hidden`，本卦、动爻、结果仍以现有文本和 aria 标签读取；
- 中断、页面隐藏或再次起卦时必须清理计时器与临时 class，不能遗留半成状态。

## 验收

1. 桌面和 375px 宽度下无裁切、无横向滚动；
2. 正常动效完整发生且不超过 1.55 秒；
3. 减少动态效果直接展示最终结果；
4. 动爻朱砂只出现一次，朱砂不扩散到其他 UI；
5. 起卦、确认、重试、食录和新食堂/外出推荐无回归；
6. 浏览器控制台无错误，并完成 Pages 公网验收。
