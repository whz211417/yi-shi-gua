# 易食卦

一个可离线打开的“今天吃什么”小网页。它把报数起卦的仪式感与日常饮食的去重逻辑结合起来：卦象只负责让选择更有趣，实际推荐会尽量避开最近重复的餐品、主食和主要蛋白质。

## 打开方式

> **不要直接双击 `index.html`。** 页面使用 ES Modules；在 Windows 的 `file://` 直接打开模式下，Chrome 会出于安全限制拦截模块脚本，导致按钮看起来存在但不会响应。

请在项目根目录启动一个本地静态服务器：

```bash
python -m http.server 4173 --bind 127.0.0.1
```

然后用 Chrome 打开 `http://127.0.0.1:4173`。

## 使用方法

1. 选择早餐、午餐或晚餐。
2. 选择在学校、校外或不想走远。
3. 选择天气状态，可直接保持“不考虑”。
4. 输入 1-64 的数字，或点“随机”。
5. 点击“拈数起卦”。
6. 看到结果后，可选择“此餐可取”或“再演一卦”。
7. 确认的餐会进入“今日食录”，并影响下一餐的推荐。每一餐都可撤销。

## 自定义菜单

点击右上角“整饬食单”可：

- 新增餐品；
- 修改餐品名称、档口、来源、餐别及营养标签；
- 停用不想被推荐的餐品；
- 删除餐品；
- 恢复预装的起始菜单（共 80 项，可按自己的常去档口继续调整）。

菜单和已确认的用餐记录只会写到当前浏览器的本地存储中，不会上传到网络。清除该浏览器站点数据后，这些本地记录会被清除。

## 手机使用

### 同一 Wi-Fi 下直接用手机打开

1. 让电脑和手机连接同一个 Wi-Fi。
2. 在本项目根目录运行：

   ```bash
   python -m http.server 4173 --bind 0.0.0.0
   ```

3. 在 Windows 命令行运行 `ipconfig`，找到当前 Wi-Fi 的 **IPv4 地址**，例如 `192.168.1.23`。
4. 在手机浏览器访问：`http://192.168.1.23:4173`。
5. 如浏览器提供“添加到主屏幕”，可添加为桌面图标，之后打开更方便。

> 不要在手机上输入 `localhost:4173`：它只代表手机自己，而不是电脑。手机与电脑的本地菜单、食录分别保存在各自浏览器中，互不共享。

### 长期使用

当前项目没有配置公开部署。后续可部署到任意静态托管服务，并将网页添加到手机主屏幕；若要做成完整 PWA，也可以在下一版补齐离线缓存与应用图标。

## 推荐方式

推荐会优先：

- 避免最近两餐出现相同餐品、主食或主要蛋白质；
- 当前餐补足近期较少的蔬菜；
- 在连续出现同类蛋白质时换一种来源；
- 前一餐偏油或重口时，优先考虑较清淡或有汤的组合；
- 严格匹配餐别、所在地点和已停用的菜单项。

如果所有可选项都和最近的餐相近，网页会放宽“避重”限制并明确说明。

## 说明

- 食卦、卦辞和报数均为娱乐化交互，不用于预测或命理判断。
- 营养相关文字是日常饮食提示，不构成医疗、疾病饮食、减脂或增肌建议。
- 预装餐品仅是可编辑的起点，不代表真实食堂或餐馆的供应情况。
