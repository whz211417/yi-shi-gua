# 易食卦

一个可离线打开的“今天吃什么”小网页。它把报数起卦的仪式感与日常饮食的去重逻辑结合起来：卦象只负责让选择更有趣，实际推荐会尽量避开最近重复的餐品、主食和主要蛋白质。

## 打开方式

最简单：直接双击 `index.html`，用现代浏览器打开即可。

如果你想通过本地地址预览，在项目根目录运行：

```bash
python -m http.server 4173
```

然后打开 `http://localhost:4173`。

## 使用方法

1. 选择早餐、午餐或晚餐。
2. 选择在学校、校外或不想走远。
3. 选择天气状态，可直接保持“不考虑”。
4. 输入 1-64 的数字，或点“随机”。
5. 点击“起餐卦”。
6. 看到结果后，可选择“就吃这个”或“换一卦”。
7. 确认的餐会进入“今日食录”，并影响下一餐的推荐。每一餐都可撤销。

## 自定义菜单

点击右上角“管理菜单”可：

- 新增餐品；
- 修改餐品名称、档口、来源、餐别及营养标签；
- 停用不想被推荐的餐品；
- 删除餐品；
- 恢复预装的起始菜单。

菜单和已确认的用餐记录只会写到当前浏览器的本地存储中，不会上传到网络。清除该浏览器站点数据后，这些本地记录会被清除。

## 推荐方式

推荐会优先：

- 避免最近两餐出现相同餐品、主食或主要蛋白质；
- 当前餐补足近期较少的蔬菜；
- 在连续出现同类蛋白质时换一种来源；
- 前一餐偏油或重口时，优先考虑较清淡或有汤的组合；
- 严格匹配餐别、所在地点和已停用的菜单项。

如果所有可选项都和最近的餐相近，网页会放宽“避重”限制并明确说明。

## 说明

- 食卦、卦辞和报数均为娱乐化交互，不用于预测或命理判断。
- 营养相关文字是日常饮食提示，不构成医疗、疾病饮食、减脂或增肌建议。
- 预装餐品仅是可编辑的起点，不代表真实食堂或餐馆的供应情况。
